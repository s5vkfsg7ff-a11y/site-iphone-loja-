Landing Page - Loja Apple Loovi Iphones
Esta é a landing page oficial da loja Loovi Iphones, especializada em produtos Apple. A página apresenta os produtos disponíveis, formas de contato e informações da loja.

Funcionalidades
Apresentação de produtos Apple
Seção de contato com WhatsApp
Informações sobre a loja (endereço, horário, etc.)
Design responsivo


Tecnologias Utilizadas
HTML5 e CSS3
Hospedagem na Vercel
GitHub para controle de versão

Como Rodar Localmente
Clone este repositório.
Abra o arquivo index.html no navegador.


Observações
Este projeto foi desenvolvido para uso real da loja. Partes do código foram automatizadas com IA (Claude Code), mas todo o layout e as funcionalidades foram personalizadas para atender ao cliente.
Com esse formato, quem acessar entende rapidamente que é um projeto real, mas feito com cuidado e transparência.
